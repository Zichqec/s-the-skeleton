English:
Thank you for installing FLUX player! This ghost is a spiritual successor to the ghost FLELE, which you can find here: http://navy.nm.land.to/ukgk/flele/beginner.html
FLUX can play music for you, and save playlists. You can install different shells to change the look of FLUX, and some shells may come with custom dialogue/functions! I keep a list of shells for FLUX on my website: https://zichqec.github.io/s-the-skeleton/flux.html
FLUX is also backwards compatible with FLELE shells! As long as FLELE is closed, you should be able to drag and drop them onto FLUX to install them. Here's a collection of FLELE shells, but there are certainly more out there: https://www.deviantart.com/zichqec/favourites/87935078/flele-shells
Every shell will have it's own hotspots that you can use for music controls. These should be detailed in their readme, which you can bring up at any time by pressing F1 and changing the tab to 'Shell information...'

If you find any bugs, or FLELE shells that aren't compatible, please get in touch with me! I can be reached on any of my socials, linked at the bottom of my website here: https://zichqec.github.io/s-the-skeleton/
If you find an error in the English text for FLUX, you can reach me at that same link. And if you're interested in translating FLUX to your language, please check out translating_flux.txt in FLUX's folder!

If you want to make your own shell for FLUX, check out the documentation in the make_your_own_shell folder within FLUX's files! _how_to_make_a_shell.txt has a guide on basic shell creation, as well as documentation on FLUX's special setup for custom functions/dialogue.

Many thanks to the folks who have translated FLUX!

Русский:
Спасибо за установление Плеер FLUX! Этот приведение духовный наследник к FLELE, который можешь найти здесь: http://navy.nm.land.to/ukgk/flele/beginner.html
FLUX имеет играть для тебя музыку, и сохранить плейлисты. Тебе можешь скачать другие внешние видов, чтобы изменить вид FLUX, и некоторые из этих есть своему личному разговоры/функцами! У меня список Видов на моей сайте: https://zichqec.github.io/s-the-skeleton/flux.html
Он тоже есть обратной совместимость с видов для FLELE! Только если FLELE закрыто, он должен разрешить тебе перетаскивать их на его чтобы загрузить. Вот одна коллекция видов, но конечно, много еще в интернете есть: https://www.deviantart.com/zichqec/favourites/87935078/flele-shells
У каждого вида свои горячие-точках, который можешь использовать для контроля плеера. Ты можешь эти найти в README, и чтобы открыть его, нажми кнопку F1 и изменить вкладка в "Информация обо Внешном Виде..."

Если найдешь какие-нибудь ошибки, или виды FLELE что не рвботает, пожалуйста, сообщите мне! Можно сообщить мне на любие из моих социальных сетях, который есть здесь на нижнем стороне моего сайта: https://zichqec.github.io/s-the-skeleton/
Если найдешь какие-нибудь ошибки в русском перевода, пожалуйста, сообщить переводчик на https://ev-n-learning.tumblr.com . И если хочешь переводить в своему языка, смотри на translating_flux.txt в папка у FLUX!

Если хочешь добавить свой собственный Внешний Вид для FLUX, получишь поглядеть на папке make_your_own_shell внутри файли FLUX! У _how_to_make_a_shell.txt справочник на основнание создание видов, и также документация на особенный настрой FLUX для особенные функций/разговоры.

Если когда-нибудь разгрузить вид и FLUX дает тебе ошибок, ничего не делает, перейти на файл SSP/ghost/flux/ghost/master/fx_shelldialogue.dic, удалить все содержание и сохранить, а перезагрузить FLUX. Потом, перейти в Меню Конфигурации и выключить особенный разговори Вида. Твой вид потом работает как обычный вид, и это будет мешать ошибка. Ты можешь включить если сменишь в другой (работает) вид. И расскажи автора, что их вид не работает! URL автора должен быть где-то в меню правой кнопкой мыша.

Большое спасибо всех переводчикам FLUX!

Русский перевод: Smoky https://smokycinnamon.github.io/indifferentsorrel/
