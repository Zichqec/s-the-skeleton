Thank you for downloading Ukatranslator! This simple ghost is made to help you run the dialogue of other ghosts through machine translators, by cleaning out most of the sakurascript tags. It can copy dialogues to your clipboard, or even automatically open them in either DeepL or Google Translate's online translators! It's not perfect by any means, and certainly not a replacement for real translation, but it can give you the gist of what the other ghost is saying!

Please note: Ukatranslator makes use of a lot of features that were added to SSP after 2.5, and was released for version 2.5.33 specifically. Please ensure your SSP is up to date! (And thank you Ponapalt for working with me to make this ghost possible!)

Make sure the ghost you want to translate is open, then open Ukatranslator's menu and pick the ghost you want from the list. Tell it to start listening, and it'll react any time the other ghost speaks. What it does will depend on your settings; by default, it will ask you what you want it to do each time.

You can make Ukatranslator invisible with a dressup labelled 'Hide' if you want it to be completely out of the way. You can right click the tray icon at any time to remove the dressup and display the translator again.

If you have any ideas for features that would improve the translator, please do let me know! I hope that it will be a good tool to help bridge the gap between different language ukagaka communities, and help spread ukagaka to new language communities as well.

On that note, if you want to translate Ukatranslator's menus into another language, it's simple to do so! Copy the ut_lang_en.dic file in Ukatranslator's ghost>master folder, rename it with the language code for your language, and add an entry in the yaya.txt file with the name of the file you just made. Then, open the file you made and translate the dialogue inside. (If you've never translated a ghost before, this guide should be helpful to you: http://ashido.com/ukagaka/translation.html ) The file should contain everything you need to translate and no more. If you run into any issues, let me know! When you've completed your translation, you can contact me to send me the file. My links are at my website below. I greatly appreciate any and all translations!

You can contact me for any questions/bug reports/translation submissions from the links at the bottom of my site: https://zichqec.github.io/s-the-skeleton/
You can also find me on the Ukagaka Dream Team Discord server! https://ukagakadreamteam.tumblr.com/discord

Danke fürs herunterladen von Ukatranslator! Dieser einfache Ghost ist dazu da dabei zu helfen Dialog von anderen Ghosts von maschinellen Übersetzern übersetzen zu lassen, indem er die meisten Sakuraskript-Tags entfernt. Er kann Dialoge in die Zwischenablage kopieren oder sogar automatisch in entwerder DeepL's oder Google Translate's Online-Übersetzer öffnen! Das ist auf keinen Fall perfekt und sicher kein Ersatz für menschliche Übersetzung aber man kann so zumindest einen Eindruck davon bekommen was ein anderssprachiger Ghost sagt!

Bitte beachte: Ukatranslator verwendet einige Funktionen, die zu SSP erst nach Version 2.5 hinzugefügt wurden, und wurde speziell für Version 2.5.33 veröffentlicht. Bitte vergewissere dich, dass SSP auf dem neuesten Stand ist! (Und Danke an Ponapalt für die großartige Zusammenarbeit um diesen Ghost zu verwirklichen!)

Vergewissere dich, dass der Ghost, der übersetzt werden soll, geöffnet ist, dann öffne Ukatranslator's Menü und wähle den Ghost aus der Liste aus. Sag Ukatranslator, dass er anfangen soll zuzuhören und er wird jedes mal reagieren, wenn der Ghost spricht. Was Ukatranslator genau macht hängt von den Einstellungen ab; Standardmäßig wird er jedes Mal nachfragen was er tun soll.

Man kann Ukatranslator mit einem Dressup namens 'Hide' unsichtbar machen, wenn er komplett aus dem Weg sein soll. Indem man das Icon im Infobereich rechts-klickt und das Dressup wieder entfernt, kann er jederzeit wieder sichtbar gemacht werden.

Falls dir irgendwelche Verbesserungsvorschläge für Ukatranslator einfallen, lass mich bitte davon wissen! Ich hoffe sehr, dass Ukatranslator ein gutes Werkzeug dafür sein wird die verschiedensprachigen Ukagaka Communities zusammenzubringen und neue andersprachige Ukagaka Communities zu gründen.

In diesem Sinne, wenn du Ukatranslator's Menüs in eine andere Sprache übersetzen möchtest ist das ganz einfach! Öffne dann die Datei, die du erstellt hast, und übersetze den Dialog darin. (Falls du noch nie einen Ghost übersetzt hast sollte diese Anleitung sehr hilfreich sein: http://ashido.com/ukagaka/translation.html ) Diese Datei sollte alles beinhalten, dass du übersetzen musst.
