SAORIs

SAORI are programs that extend the functionality of ghosts! You can include these files in your ghost, and call them from the ghost's code to do things that the ghost's coding language can't do on its own. There are two types of SAORI: SAORI-Basic, and SAORI-Universal. SAORI-Basic are .exe files, and must be called using a proxy SAORI if you're writing in AYA/YAYA. SAORI-Universal are .dll files, and can be called without needing the proxy.


ImageCompositor

Download it here!

This SAORI can be used to take multiple images and composite them together. It was created for Warrior Generator, in order to export generated designs with a transparent background, but it may be useful for other things if you need some simple image compositing!

Plugins

Plugins are like SAORI, but for SSP instead of individual ghosts! They extend the functionality of SSP, and can do all kinds of things. Most plugins will have a menu you can access by right clicking a ghost, going to the plugin tab, and selecting that plugin.


Weather Station

Download it here!

This plugin is both for users who want weather data, as well as developers who would like their ghost to react to weather data! By using this plugin to gather weather info, users will only need to put in their location once, and then every compatible ghost will have the weather! Built using weatherapi.com, this is using a proper API, so it should be a lot more stable than previous weather implementations. I only get 1 million free API calls a month, though! If that limit worries you, the code is open source and you are more than welcome to take it and implement it in your own ghost! Please credit me if you use my code, and link back to my site here.

Tools

I'm considering Tools to be any external programs that I create to aid ghost dev. This is a broad category, so be sure to read the readmes carefully!


olddef2newdef

Download it here!

Want to change over to the New Definition of SERIKO to get the fancy new options, but don't want to spend hours reorganizing your shell code? This little program has you covered! Simply drag and drop your surfaces.txt file onto the executable, and it will convert your code for you! It will also save a backup of the original code, in case anything should go wrong.


CommuHelper

Download it here!

Looking to write lots of ghost communication conversations, but tired of the tedious formatting? Worry no more! With CommuHelper, you can easily write long conversations and have machines do all the tedious formatting for you! CommuHelper is set up with particular formatting in mind, so please read the readme carefully!
