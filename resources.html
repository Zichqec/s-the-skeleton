Programs & Plugins

  • Tama - Debugging tool for AYA/YAYA (Dark theme mod by me)
  • Coordin - Easy collision maker
  • png2pna - Converts png files to pna files
  • findwin - program + saori for letting ghosts see open programs
  • SEED - Shiori Event Emulator for Developers
  • Discord Plugin - Shows current ghost in Discord
  • Balloonmaker - Plugin that lets you create a simple balloon through a menu
  • easydelete - Easily add many files to delete.txt
